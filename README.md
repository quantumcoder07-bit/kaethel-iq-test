# kaethel-iq-test
# K-Cognitive Engine v1.0
> **Project:** Kaethel Intelligence Assessment | **Status:** Live

A minimalist, high-speed IQ assessment tool designed to measure cognitive processing speed and logical accuracy. Built for the **Kaethel** ecosystem.

## 🧠 Core Logic
The engine utilizes a weighted algorithm to calculate a real-time IQ estimate based on accuracy and temporal efficiency:

$$IQ_{est} = 80 + (Correct \times 5) + \text{TimeBonus}$$

* **Baseline:** 80 IQ
* **Accuracy:** +5 points per correct response.
* **Temporal Bonus:** Calculated if completed in under 300 seconds.

## 🛠️ Tech Stack
- **UI:** Tailwind CSS (Glassmorphism design)
- **Engine:** Vanilla JavaScript
- **Typography:** Space Grotesk

## 🚀 Deployment
Hosted via GitHub Pages. Data is processed entirely client-side for maximum privacy.
